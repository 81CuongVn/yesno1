YesNo is a simple open-source Discord yes/no poll bot for created by advaith. It is based on the YesNo bot built in to Google Allo, which shut down in March 2019.

YesNo is super easy to use and requires no setup. To create a poll, mention the bot (@YesNo) and ask the question. For example: @YesNo is YesNo the best bot?. Polls will be pinned to the channel so people can easily find them.

YesNo now supports Slash Commands! Instead of sending messages to use YesNo's commands like close, you can press / and use the commands from your Discord client!

You can close a poll (which will unpin it and send the results) by using /close.
